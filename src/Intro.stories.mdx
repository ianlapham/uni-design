# Get started

Uniswap Design System is a reusable component library for building web3 interfaces.

## Install

Uni design components are written in React, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df). It requires Storybook version 5.2-beta and up.

### **Use**

Import components you want into your UI

`import { Button } from '@uniswap/design-system';`

and use them like so

```
  const example = () => (
    <div>
      <Button appearance="primary" onClick={() => 0}>Do something</Button>
    </div>
  )
```

### **Run and develop components locally**

`yarn && yarn run storybook`
